Descripción
Aplicación desarrollada en JavaScript utilizando Programación Orientada a Objetos y XMLHttpRequest para consumir datos desde la API:

https://jsonplaceholder.typicode.com/users

La aplicación permite:

Listar todos los usuarios.
Buscar información básica de un usuario.
Mostrar la dirección completa de un usuario.
Mostrar información avanzada de un usuario.
Listar compañías y catchphrase.
Ordenar usuarios alfabéticamente.
Objetos de la aplicación
Clase Usuarios
La clase Usuarios se encarga de:

Obtener los datos desde la API.
Almacenar la información de los usuarios.
Gestionar las búsquedas y listados solicitados.
